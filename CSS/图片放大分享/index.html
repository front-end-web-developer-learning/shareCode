图片的缩放分享

要求：鼠标悬浮，图片放大

css实现方式

  • 影响布局
    • 使用zoom属性(百分比缩放)，但是存在兼容性问题，示例见demo1
    • 在确定尺寸时，直接修改尺寸(非百分比缩放，直接设置宽高)，示例见demo2
  • 不影响布局
    • 使用css3的scale(百分比缩放)，示例见demo3

相比而言，使用scale
  • 兼容性较好，而且不影响布局
  • 可以不用事先知道图片的尺寸，缩放比例、缩放中心灵活把控
  • zoom和修改尺寸都会导致整个页面布局的变动，重新渲染，scale只是在原来的元素上重绘，性能上会好些
