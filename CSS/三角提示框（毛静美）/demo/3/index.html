纯CSS实现边上带小三角的div提示框

1.带边框的提示框

     /*
      * 1. 首先画一个普通的 div 框
      * 2. 再在适当的位置放一个小三角形
      * 3. 再在三角形的内部再放置一个小三角(border宽度一样)，进行定位，形成边框，并遮住多余的边
      */

      <div class="tipComponent">
        <div class="rectangle">
            css实现边上带小三角的提示框
        </div>
        <div class="triangle">
            <div class="triangle_inner"></div>
        </div>
      </div>

      .rectangle{
        width: 400px;
        height: 100px;
        border: 2px solid black;
        background: red;
      }
      .triangle{
        width: 0px;
        border:20px solid transparent;
        border-top-color: #000;
        left: 195px;
        position: absolute;
      }
      .triangle_inner{
        width: 0px;
        border:20px solid transparent;
        border-top-color: red;
        left: -20px;
        top:-23px;
        position: absolute;
      }
