解析border，轻松实现CSS画三角形

关于border

我们经常使用的 border 最初状态如下所示：

      <div class="border"></div>

      .border{
        height: 50px;
        border: 10px solid #000
      }

but... 当我设置了其他边的边框但是显示为透明时

        <div class="border-aside-transparent"></div>

        .border-aside-transparent{
          height: 50px;
          border: 10px solid transparent; 
          border-top: 6px solid #000;
          background:#ccc;
        }
上边框变成梯形了！这是怎么回事？

在未设置left和right的时候便不会有此问题，top妥妥的矩形，若都设置了只能相交部分平分一半了，如下所示：

        <div class="border-aside-all"></div>
        .border-aside-all{
          height: 50px;
          border-top: 10px solid rgb(238, 96, 40);
          border-right: 10px solid rgb(57, 69, 238);
          border-bottom: 10px solid plum;
          border-left: 10px solid green;
          background:#ccc;
        }

四边border的位置，很明显了对不对？接下来就可以画三角形啦啦啦！！！

CSS画三角形

      <div class="border-square"></div>
      .border-square{
        width: 0px;
        border: 20px solid transparent; 
        border-top-color:  #000;
        <!-- background: #ccc;  -->
      }
