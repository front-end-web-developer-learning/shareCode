在清除浮动的时候，有一种做法是在父元素内部增加新的元素，应用clear:both清除浮动。

    style
        /* 父元素样式 */
        .parent{
            border: #007bff solid 1px;
            padding: 10px;
            width: 200px;
        }
        /* 子元素样式 */
        .child{
            float: left;
            border: 1px solid #007bff;
            padding: 5px;
            margin: 0 10px;
        }
        /* 清楚浮动元素的样式 */
        .clear{
            clear:both;
        }

    html
        <div class="parent">
            <div class="child">child</div>
            <div class="child">child</div>
            ...
            <div class="clear"></div>
        </div>

使用伪元素可以减少dom节点数，将新增的元素清除浮动部分放到父元素的伪元素:after

    style
        /* 父元素样式 */
        .parent{
            border: #007bff solid 1px;
            padding: 10px;
            width: 200px;
        }
        /* 父元素增加伪元素利用clear:both清除浮动 */
        .parent::after{
            content: "";
            display: block;
            clear:both;
        }
        /* 子元素样式 */
        .child{
            float: left;
            border: 1px solid #007bff;
            padding: 5px;
            margin: 0 10px;
        }

    html
        <div class="parent">
            <div class="child">child</div>
            <div class="child">child</div>
            ...
        </div>

效果展示

child

这两种方式的效果是一样的，使用伪元素的方式会减少dom节点数
