CSS实现各种形态的三角形

  1. 实心三角形
    • 等腰三角形 
                    /*
               * 1. 将div的宽度设为0，高度默认就是0
               * 2. 统一设置border，并全部透明显示
               * 3. 根据所需三角形旋转的情况单独设置任一边的颜色，如下：
               */

               <div class="equilateral-triangle"></div>

                .equilateral-triangle{
                  width: 0;
                  border:20px solid transparent;
                  border-bottom-color: blueviolet;  //尖角朝上（如本例所示）
                  border-top-color: brown;  //尖角朝下
                  border-left-color: brown;  //尖角朝右
                  border-right-color: brown;  //尖角朝左
                }
    • 等腰直角三角形 
                    /*
               * 1. 将div的宽度设为0，高度默认就是0
               * 2. 统一设置border，并全部透明显示
               * 3. 只要将任意两个相邻边的颜色设置出来就好了( 正方形切一半肯定是等腰直角三角形 )
               */

               <div class="isosceles-right-triangle"></div>

               .isosceles-right-triangle{
                  width: 0;
                  border: 20px solid transparent;
                  border-left-color: black;
                  border-bottom-color:#000;
                }
    • 等边三角形 
                    /*
               *前两个条件和之前一样，主要是单独设置的border宽度与统一的不一样就行，大小根据需求而定
               */

               <div class="triangle"></div>

                .triangle{
                  width: 0;
                  border: 20px solid transparent;
                  border-bottom: 50px solid #000;
                }
  2. 有边框的三角形 
              /*
           * 思路：将2个三角形叠加起来，外层的三角形稍微大一些。
           */

           <div class="outside-normal">
              <div class="inside-normal"></div>
           </div>

           .outside-normal{
            width: 0;
            border:50px solid transparent;
            border-bottom-color: #000;
            position: relative;
            background: #ccc;
          }
          .inside-normal{
            width: 0;
            border:40px solid transparent;
            border-bottom-color: red;
            position: absolute;
            background: green;
          }
        
              /*
           * 思路：将上面图形进行设置定位
           */

           <div class="outside-over">
            <div class="inside-over"></div>
           </div>

           .outside-over{
            width: 0px;
            border:100px solid transparent;
            border-bottom-color: #000;
            position: relative;
          }
          .inside-over{
            width: 0px;
            border: 90px solid transparent;
            border-bottom-color: #fc0;
            position: absolute;
            top: -85px;
            left: -90px;
          }
  3. 空心三角形 
              /*
           * 原理：两个三角形重叠，下面的边重合
           */

           <div class="empty">
            <div class="empty-inside"></div>
           </div>

           .empty{
            width: 0;
            border-left: 30px solid transparent;
            border-right: 30px solid transparent;
            border-bottom: 30px solid #333;
            position: relative;
            background: #ccc;
           }
          
          .empty-inside{
            width: 0;
            border-left: 28px solid transparent;
            border-right: 28px solid transparent;
            border-bottom: 28px solid #F0981C;
            position: absolute;
            left: -28px;
            top: 2px;
          }
    如果只想要箭头？ 
              /*
           * 原理：将里面的形成的三角形 border-color 设置成想要的背景颜色即可
           */

           <div class="empty-arrow">
            <div class="empty-arrow-inside"></div>
           </div>

           .empty-arrow{
            width: 0;
            border-left: 30px solid transparent;
            border-right: 30px solid transparent;
            border-bottom: 30px solid #333;
            position: relative;
            /* background: #ccc; */
           }
          
          .empty-arrow-inside{
            width: 0;
            border-left: 28px solid transparent;
            border-right: 28px solid transparent;
            border-bottom: 28px solid #fff;
            position: absolute;
            left: -28px;
            top: 2px;
          }
